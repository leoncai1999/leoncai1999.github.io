Texas Running Club App

Using XCode and Swift, I built an iOS application for our intramural running club. The app is inspired by the website but also incorporates additional features. Main features:

• A poll to vote on the route we run everyday. The poll is connected to a Firebase Database which is used to track member votes and select a route.
• A map which displays all of the routes we run as polyline objects.
• A feed showing when members ran and how long/fast they ran. The feed is populated by Strava's API for club activites.
• A news feed of tweets from running related Twiiter accounts powered by Twitter API.

The app also includes resources for joining the email list and accessing the club's social media accounts.

Workout Heatmap Web App

Using React, HTML/CSS and Javascript, I built a Web App that connects with a user's Strava account to visualize and analyze their workout data. App Features:

• The user can view all of their outdoor workout activities in an interactive heatmap. A panel allows the user to filer activities by sport, workout type, number of members and time of day. Additionally, the user can recenter the map to the different locations of their workouts.
• All of the user's activities are also displayed in a table that can be sorted by a variety of attributes. If desired, this table can be downloaded to Excel.
• The Stats page provides additional workout analytics not provided by Strava including a breakdown of average mileage and effort exerted by day, the most common times of day in which workouts are started, and the amount of activity time spent in each heart rate zone.
• The app's algorithm scans all of the user's map plotted activities to determine which routes the user has repeated and how many times those routes have been repeated. Then, the user's most frequented routes are displayed.

ApartmentGate Android App

Using Android Studio and Kotlin, a friend and I built an Android App from scratch that allows users to browse apartment listings in San Francisco. App Features:

• The user is prompted to create a profile, providing details about their work address, commute method, budget, and neighborhood preferences. Apartment listings are then graded based on how closely they fit the user's criteria.
• The listings, retrieved from Apartments.com, and user profiles are stored in a Cloud Firestore database. Additionally, information about the popularity of each neighborhood and average rent is stored.
• When an individual listing is clicked, additional details about the property are shown, the user's commute from that apartment is shown on the map, and additional details of the neighborhood are provided via the WalkScore, SoundScore and Google Maps APIs.

CollegeFitForMe Web App

Using React, HTML/CSS and Javascript, a group of peers and I built a dynamic website that allows High School students to browse prospective colleges and learn more about them. App Features:
